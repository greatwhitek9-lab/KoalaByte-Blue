# KoalaByte Blue — Case Print Files

This branch hosts the downloadable STL ZIP for the KoalaByte Blue V2 Heltec Edition case.

## Download

Use `koalabyte_blue_v2_8button_case_print_files.zip` from this branch.

## Contents

- Hollow rear shell with koala ears and one top-center antenna port
- Front door with 8-button layout and swapped DB9 / HDMI / micro USB port row
- Lower and upper removable shelves
- ESP32-S3 DualEye internal mount
- 8-button internal mount bar
- Heltec T114 screen-forward mount
- Print notes and case parameters

## Print starting point

FLSUN V400, PLA+, 0.20 mm layer height, 4 walls, 25–35% infill, tree/organic supports where needed, and a 5–8 mm brim for tall shell parts.

These are prototype fit-check print files. Test-fit all hardware before long prints.
