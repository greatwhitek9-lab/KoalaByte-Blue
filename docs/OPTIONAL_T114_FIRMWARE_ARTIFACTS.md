# Optional T114 Firmware Artifacts

This document preserves the useful findings from the optional T114 build/flash firmware scripts mined from `Backup_heltec` and defines how their artifacts are now generated in the KoalaByte Blue V2 Heltec Edition default paths.

## Default path status

The HCI USB, color-mouth, GNSS, face-state, and passive BLE protocol artifacts are now generated by default through:

- `scripts/install_pi.sh`
- `bash scripts/flash_all_components.sh --install-firmware`, because it runs `scripts/install_pi.sh`
- `bash scripts/flash_all_components.sh --all`, because it runs `scripts/install_pi.sh`
- `scripts/build_firmware_all.sh`
- `.github/workflows/koalabyte-blue-ci.yml`

The default helper is:

```bash
bash scripts/build_default_t114_protocol_artifacts.sh
```

It writes:

```text
logs/optional_t114_firmware_artifacts.json
logs/t114_2g4_antenna_status.json
```

This helper records the mined protocol artifacts but does not automatically flash a new T114 firmware profile.

## Preserved artifact names

The HCI USB build metadata artifact is:

```text
logs/t114_hci_usb_mode.json
```

The HCI USB active-mode artifact is:

```text
logs/t114_active_ble_mode.json
```

The mined build helper name was:

```text
build_nrf52840_t114_hci_usb.sh
```

The flash-helper mismatch found during mining is retained as a warning: the mined flash script referenced `scripts/build_koala_konnect_t114.sh`, but the useful build helper was `scripts/build_nrf52840_t114_hci_usb.sh`.

## Preserved protocol names

These protocol message types are retained in docs and generated manifests:

```text
killerkoala_face
gnss_nmea
ble_adv_seen
```

They represent the face-state command, GNSS forwarding event, and passive observation event used by the mined T114 firmware profiles.

## Guardrail

Do not add optional T114 HCI USB or Heltec color-mouth firmware automatic flashing to the default path until the exact board target, firmware role, and recovery process are validated on hardware.

Protocol artifacts are default. Automatic firmware flashing is still guarded.
